# Mr. Skeletal

A JSON structure stripper and data 
extractor for LLM conversation exports.

## What It Does
- Strips JSON structure to reveal skeleton
- Extracts clean conversation data from exports

## Supported Platforms
- Claude ✅
- Gemini ✅
- ChatGPT ✅

## Files
- mr_skeletal.py - strips JSON structure
- claude_extractor.py - extracts Claude data

## Data Extracted From Claude
- Role (normalized)
- Message text
- Timestamps
- Response times
- Files and attachments

Independent AI researcher with 15+ months
